X_Tom
=====

雷霆战机简易版

本程序模仿Z-type改造成了一个打飞机游戏，感觉效果还不错，只是每一个关卡的过渡比较简单粗暴。
This is a game imitating from <a href=http://phoboslab.org/ztype/>Z-type</a>. I think the java GUI is ok.
